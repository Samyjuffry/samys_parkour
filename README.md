# Samy's Parkour
A Pygame-based 2D platformer game.
## Description
Samy's Parkour is a 2D platformer game where the player controls Samy to run, jump, navigate through obstacles, and avoid hazards. Each level goes by it increases in difficulty using tighter platforms, enemies, and environmental dangers, leaving a tiny bit of room for errors.
## How to run the game
1. Install Python 3.13
2. Install Pygame `pip install pygame`
3. Extract all files
4. Run platformer.py
## Controls
Arrow keys to move the character.
